# Credit Scoring Model — CodeAlpha Internship

## Objective
Predict creditworthiness using financial and behavioral data.

## Dataset
Hybrid synthetic dataset (20,000 records).

## Features
Income, debt, utilization, payment history, resilience, dependency ratio.

## Models
Logistic Regression, Random Forest, Gradient Boosting,
Blended Ensemble, Stacked Ensemble.

## Results
Blended ROC-AUC ≈ 0.97  
Stacked ROC-AUC ≈ 0.97  
No overfitting detected.

## Conclusion
Ensemble learning improves robustness and generalization.
